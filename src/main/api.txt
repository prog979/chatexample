GET api/auth    - получение авторизации пользователем
Запрос -
Ответ - {
            "result": true
            or
             "result": false
}

GET /api/users      - получить список пользователей
Запрос - пустой
Ответ:  - {
    "users":[
            {
               id: 1345
               user: "Иван Иванов"
            },
            {
                id: 1845
                user: "Сидор Сидоров"
            }
            ....
            ]}


GET /api/messages   - получить список сообщений
Запрос - пустой
Ответ:  {
            "messages": [
                {
                    "name": "Имя юзера",
                    "time": "11:15 01.01.2021",
                    "text": "Текст сообщения"
                },
                ...
            ]
        }

POST /api/messages  - отправка сообщения
Запрос  - text       - текст сообщения
Ответ   -   {
                result = true
            }

POST /api/users     - добавление нового пользователя при логине

Запрос - name   - имя пользователя
Ответ   -   {
                result = true
            }
